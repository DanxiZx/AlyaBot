> 📲 Ideal para ejecutar el bot en dispositivos móviles. Copia y pega los comandos uno por uno

```bash
> pkg update && pkg upgrade
> pkg install git nodejs ffmpeg imagemagick -y
> git clone https://github.com/DanxiZx/madaraBot
> cd madaraBot
> npm install
> node index.js
```
